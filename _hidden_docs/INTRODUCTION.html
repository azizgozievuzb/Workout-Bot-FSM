Концепция

Что это за проект?

Telegram Mini App, где пользователь выполняет 35-минутную тренировку, снимает себя на камеру, а AI (Gemini Vision) оценивает качество выполнения.

🎯 Ключевая идея

Мотивация через живого человека. Не просто бот — а система, где кто-то реальный платит за твои тренировки, следит за прогрессом, отправляет стикеры утром и вечером, и дарит подарки за достижения. AI — помощник. Человек — мотиватор.

«Смотрящий — это не функция. Смотрящий — это отношения.»

🏗️ Платформа

Telegram Mini App (Vite + React фронтенд). Python + Aiogram бэкенд. Supabase PostgreSQL база. Gemini Vision API для анализа видео.

💰 Монетизация

3 способа оплаты: Telegram Stars, промокоды (покупка напрямую у админа) и криптовалюта. Смотрящий всегда платит за доступ Игрока. Триальный период — 1 неделя бесплатно.

Система ролей

Два экрана. Две роли.

Каждый пользователь может быть Игроком, Смотрящим или обоими одновременно. Переключение — одной кнопкой.

🏋️
Игрок
Тот, кто тренируется
  • 🎥 Выполняет тренировку на камеру (35 мин)
  • 🤖 AI оценивает качество 0–100%
  • 🪙 Зарабатывает Золото за тренировки
  • 🛒 Тратит Золото в Магазине (подарки от Смотрящего)
  • 🔥 Держит стрик (дни подряд)
  • 🏆 Может участвовать в турнирной таблице
  • 🩸 Дни отдыха — 3/мес (только женщины)
👀
Смотрящий
Тот, кто следит и мотивирует
  • 💳 Оплачивает доступ Игрока
  • 📨 Отправляет стикеры 2 раза/день
  • 💕 Заполняет Шкалу Заботы
  • 🎁 Добавляет подарки в Магазин Игрока
  • 🛡️ Покупает Стрик-щит (тайно)
  • 📦 Записывает Капсулу Времени
  • 🏅 Нажимает «Горжусь!»
  • 👥 До 3 Игроков (Премиум)

Активация
1

Самостоятельная регистрация → Смотрящий

Новый юзер открывает бота. Проходит онбординг: язык, пол, фото. Далее — выбор тарифа, оплата, генерация инвайт-ссылки.

2

Вход через ссылку → Игрок

Кто-то кликает инвайт-ссылку. Если новый — полный онбординг + анкета тренировок. Если существующий — только анкета + подтверждение.

3

Отклонение / Истечение → 50% возврат

Неважно, отклонил Игрок или ссылка просто истекла — Смотрящему возвращается 50% оплаты. Всегда одинаково.

4

Обе роли одновременно

Если Азиз — Смотрящий Боба, а Боб — Смотрящий Азиза, у обоих есть переключатель ролей. Каждый платит за своего Игрока. Тарифы могут отличаться.

Геймификация

7 механик вовлечённости

Превращают Смотрящего из пассивного наблюдателя в активного участника. Ни одну из этих механик AI заменить не может.

💕

Шкала Заботы

Ежедневно

Смотрящий отправляет 2 стикера в день (минимум 8ч между ними). Шкала заполняется за ~7 дней. При 100% — открывается возможность подарить буст.

🎁

Кастомные подарки

Магазин

Смотрящий добавляет подарки в Магазин Игрока: творческие (массаж, ужин) или реальные (протеин, гантели). Игрок «покупает» их за Золото.

🏅

Кнопка «Горжусь!»

Таблица

Смотрящий нажимает «Горжусь!» — это отображается в турнирной таблице: «Азиз гордится 3 Игроками с рейтингом 3100».

🔄

Синхро-стрик

Еженедельно

Если Игрок тренируется И Смотрящий стикерит каждый день — оба держат синхро-стрик. 7 дней → X2 буст на 1 день. Месяц → ещё X2 на 3 дня.

Капсула Времени

Эмоциональный

Смотрящий записывает текст или видео (до 30 сек), которое откроется строго через 30 дней. Нельзя отменить. Экстремально мощная привязка.

🛡️

Стрик-щит

Покупка

Смотрящий тайно покупает защиту стрика (1–7 дней, макс. неделя). Игрок узнаёт только при пропуске: «Ваш стрик спасён Смотрящим! 🛡️»

✅

AI — Единственный источник баллов

Ключевой принцип

Баллы не существуют до проверки AI. Программа ничего не начисляет автоматически — только AI решает, сколько баллов получит упражнение. Нет человека на видео или просто стоит/сидит → AI возвращает 0 → программа начисляет 0 баллов за упражнение. Это не штраф и не отбирание — баллов просто не было и не будет без реальной тренировки.

Искусственный интеллект

Как начисляются баллы

Баллы за упражнение НЕ начисляются автоматически. Единственный источник баллов — решение AI после анализа видео.

⚙️ Как это работает

Игрок записывает видео упражнения → видео отправляется на Gemini Vision → AI анализирует → возвращает число от 0 до 100 → программа начисляет ровно столько баллов. Без AI — нет баллов.

Социальность

Турнирная таблица

Участие строго по согласию (opt-in). Две категории номинаций.

🏋️ Номинации Игрока

  • 🏆 Самый сильный — макс. общих баллов
  • 🔥 Самый стабильный — длиннейший стрик
  • ⚡ Лучший за неделю — макс. за 7 дней

👀 Номинации Смотрящего

  • 💎 Самый щедрый — потратил на бусты
  • 📨 Самый заботливый — больше всех стикеров
  • 🏅 Лучший наставник — рейтинг Игроков

🩸 Дни отдыха

Игроки женского пола могут отметить до 3 дней в месяц как дни отдыха. Стрик не сбрасывается, баллы = 0 без штрафа. Смотрящий видит статус «День отдыха».

Технологии

Стек

🐍

Backend

Python — язык программирования, на котором пишется вся серверная логика (обработка команд, расчёт баллов, управление стриками).

Aiogram 3 — фреймворк для Telegram-ботов. Это «переводчик» между Telegram и нашим кодом. Принимает сообщения от юзеров, обрабатывает кнопки, отправляет ответы.

Supabase (PostgreSQL) — облачная база данных. Хранит всех пользователей, стрики, баллы, подарки, инвайты. Управляется через веб-панель, не нужен отдельный сервер.

FSM на XState — логика поведения бота описана как конечные автоматы (блюпринты в TypeScript). При разработке эти схемы переводятся в Python-код.

💡 Почему этого достаточно: Aiogram обрабатывает тысячи сообщений, Supabase масштабируется автоматически, Python — самый простой язык для прототипирования и AI-интеграций.

⚛️

Frontend

Vite — инструмент сборки (как «компилятор» для веб-приложений). Берёт наш код и превращает его в быстро загружающийся сайт. Работает мгновенно, нет задержек при разработке.

React — библиотека для создания интерфейсов. Это то, из чего строятся кнопки, экраны, анимации. Используется в Instagram, Facebook, Netflix. Самый популярный инструмент для веб-приложений.

Telegram Mini App — наш React-сайт открывается прямо внутри Telegram (без браузера). Юзер нажимает кнопку → открывается экран тренировки с камерой, таймерами, прогрессом.

💡 Vite + React — это как камера + плёнка. Vite собирает, React рисует. Вместе они создают интерфейс тренировки, который работает на любом телефоне.

🤖

AI

Google Gemini Vision API — самая мощная мультимодальная AI-модель, которая понимает видео. Мы отправляем запись тренировки → Gemini анализирует технику → возвращает оценку 0–100% за каждое упражнение.

Чисто позитивная система: AI только добавляет баллы. Нет человека = 0 баллов (не минус).

💳

Платежи (3 способа)

⭐ Telegram Stars — основной способ оплаты. Встроен в Telegram. Юзер покупает Stars и платит ими за тариф.

🎟️ Промокоды — покупаются напрямую у админа (тебя). Смотрящий получает код → вводит в боте → активирует тариф без Stars. Для VIP-клиентов или особых случаев.

₿ Криптовалюта — дополнительный способ оплаты для тех, кому удобнее платить криптой. Конвертация в подписку через бота.

💡 В любом случае: 1 неделя триал. 50% возврат при отказе. Смотрящий всегда платит за Игрока.
