# FSM-Архитектура (Blueprints) для Telegram-бота Тренировок

Этот репозиторий представляет собой "чертеж" (блюпринт) полного цикла логики бота, созданный с помощью **XState** и **Stately Studio**. Конкретного программного кода здесь нет — только логические схемы, которые в будущем послужат фундаментом для разработчиков.

## Как работать со схемами:
1. Зарегистрируйтесь в [Stately Studio](https://stately.ai).
2. Настройте двустороннюю синхронизацию (GitHub Sync) с данным репозиторием (папка `fsm-диаграммы`).
3. Визуальные изменения в Stately автоматически преобразуются в файлы XState (.json / .ts) и отправляются сюда.

## Заложенные в блюпринты механики:
* **Ежедневные тренировки** с проверкой видео-доказательств.
* **ИИ-Тренер (Gemini Vision)**, оценивающий старания на видео (>50%).
* **Геймификация**: Рейтинговые лиги, Стрики (🔥) и заработок "Звезд".
* **Магазин Поощрений**, где можно тратить звезды на призы.
