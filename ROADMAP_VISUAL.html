🏋️ Workout Bot (4G)

Telegram Mini App — тренировки с AI-оценкой

💡 Концепт проекта

Telegram Mini App для 35-минутных тренировок. Пользователь записывает тренировку на камеру телефона (16 упражнений по 40 секунд). Видео отправляется на Gemini Vision AI, который оценивает технику от 0 до 100. За баллы начисляются звёзды (внутренняя валюта) → звёзды тратятся в магазине (скипы, аватарки, лутбоксы).

Уникальная механика: система пар — у каждого Игрока есть Ответственный (родитель, тренер, друг), который мотивирует, следит за прогрессом и покупает бусты X2.

Python 3.11 Aiogram 3 FastAPI Vite + React + TS Supabase PostgreSQL XState FSM Gemini Vision API Railway

🎮 Player (Игрок)

  • Тренируется 35 мин перед камерой
  • Получает AI-оценку и звёзды
  • Копит стрики (серии дней)
  • Тратит звёзды в магазине

👀 Responsible (Ответственный)

  • Оплачивает подписку
  • Видит прогресс партнёра
  • Отправляет пинг-напоминания
  • Покупает бусты X2

🛡️ Admin

  • Управляет пользователями
  • Банит (банится вся пара)
  • Смотрит статистику
  • Загружает контент

📱 Путь Игрока

/start → Язык → Роль → Пол → Опрос → Селфи → Код пары → Оплата → Тренировка!
