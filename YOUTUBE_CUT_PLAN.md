# 🎬 YouTube Long-Workout → 16 Clips (Cut Plan)

**Идея:** скачать одно длинное full-body bodyweight видео с YouTube (30–45 мин), нарезать на 16 клипов по 8 сек.

---


### ✅ Каким должно быть видео (чеклист):

- **Длительность:** 20–45 мин
- **Стиль:** full body bodyweight / HIIT / functional, **без оборудования**
- **Инструктор:** **один человек** в кадре, средний/общий план, виден всегда (не «голова крупным планом»)
- **Фон:** студия / зал / минималистичная комната, без бегущих надписей и кучи текста на экране
- **Формат:** желательно вертикальное (Shorts длинные есть редко), но горизонталь тоже сойдёт — обрежем
- **Разрешение:** ≥720p (лучше 1080p)
- **Содержит** большинство из 16 упражнений (squats / pushups / plank / lunges / jumping jacks / mountain climbers / burpees / glute bridge / crunches / knee pushups / jump squats / side plank L+R / leg raises / superman / high knees). Нормально, если 2–3 не будет — заменим стоковыми.
- **Лицензия:** для продакшена нужна Creative Commons или официальная лицензия. Для **тестирования** — любое видео, потом заменим.

### 🎯 Хорошие каналы (там часто такие видео):

- **Roberta's Gym** — много вертикалок, чистый фон, full-body
- **Pamela Reif** — 30-min full body, чистый стиль (НО: коммерческое использование запрещено, только тест)
- **MadFit** — 20–40 мин, чистый зал
- **Caroline Girvan** — full body HIIT
- **HASfit** — простой стиль, прямая камера


## 3️⃣ Найти таймкоды для 16 упражнений

Тебе нужен список вида:
```
squats        → 02:15
pushups       → 04:30
plank         → 06:50
...
```
по одной точке начала на каждое из 16 упражнений (8-сек кусок будет вырезан **с этой точки**).

### Путь A — вручную (надёжно, ~10–15 мин)

Открой видео в плеере (VLC, QuickTime). Скрабь по таймлайну, для каждого упражнения запиши **начало момента, где упражнение уже идёт ровно** (не отсчёт, не подготовка). Шаг в 0.5–1 сек — нормально.

### Путь B — через Gemini 3.1 (быстрее, но проверь)

В Google AI Studio (gemini.google.com → Studio) загрузи `long_workout.mp4` и пришли промпт:

```
Я загрузил полную тренировку. Найди в видео временные метки начала каждого 
из этих упражнений (если присутствует). Для каждого верни ТОЧНЫЙ timestamp 
в формате HH:MM:SS, где упражнение уже выполняется правильно (не разминка, 
не объяснение). Нужно ровно по одной точке на упражнение — это будет 
старт 8-секундного клипа.

Список упражнений (ключ → русское название):
- squats → Приседания
- pushups → Отжимания
- plank → Планка
- lunges → Выпады
- jumping_jacks → Прыжки (jumping jack)
- mountain → Скалолаз (mountain climbers)
- burpees → Берпи
- glute_bridge → Ягодичный мостик
- crunches → Скручивания
- pushups_knee → Отжимания с колен
- squats_jump → Прыжковые приседания
- side_plank_l → Боковая планка слева
- side_plank_r → Боковая планка справа
- leg_raises → Подъём ног лёжа
- superman → Супермен (face-down, поднимает руки и ноги)
- high_knees → Бег с высоким подниманием колен

Если какого-то упражнения нет в видео — напиши "NOT FOUND" вместо timestamp.
Верни строго в таком формате (одна строка на упражнение):
squats → 00:02:15
pushups → 00:04:30
...
```

⚠️ Gemini может ошибаться на ±2–5 сек. Открой видео, прокликай 3–4 случайные метки — если +- ок, идём дальше; если ерунда — переходи в Путь A.

## 4️⃣ Нарезать (это сделаю я)

Когда у тебя будет:
1. Файл `long_workout.mp4` в `/Users/azizgozievmacbookpro/Projects/Workout-Bot-FSM/`
2. Список таймкодов (любой из путей A/B/C)

Просто пришли мне таймкоды одним сообщением — например:
```
squats → 00:02:15
pushups → 00:04:30
plank → 00:06:50
lunges → NOT FOUND
jumping_jacks → 00:09:10
...
```

Я в одной bash-сессии:
- Прогоню через ffmpeg: на каждый ключ — `ffmpeg -ss <ts> -i long_workout.mp4 -t 8 -vf "scale=720:1280:force_original_aspect_ratio=cover,crop=720:1280" -c:v libx264 -preset medium -crf 23 -movflags +faststart -an demos/<key>.mp4`
- Положу 16 файлов в `frontend/public/demos/`
- Те, что NOT FOUND, помечу — добавишь стоковые
- `git add frontend/public/demos/` → коммитну → пушну

**Звук** на этом этапе уберу (`-an`) — в видео-подложке YT-инструктор обычно говорит / музыка чужая. Музыку и голос-пинги добавим отдельно в финале (см. `STOCK_VIDEO_SEARCH.md` → раздел «Музыка / голос»).
