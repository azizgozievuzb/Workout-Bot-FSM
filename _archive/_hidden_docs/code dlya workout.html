Техническое описание:

  • UserID / TG_ID: Внутренний GUID и Telegram ID.
  • R → P: Кому данный юзер является Ответственным.
  • P → R: Кто Ответственный у данного юзера.
  • BuyCodes: Общий объем покупок (Stars/Crypto).
  • Invite Code: Код входа или последнего продления (Renewal).
⚙️ LOGIC: Если R→P = 0, функции управления заблокированы. Доступна только закупка кодов.
